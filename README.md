# 🖼️ Aplikasi Pengolahan Citra Digital

Aplikasi web interaktif untuk pengolahan citra digital menggunakan **Streamlit** dan **Python**.  
Dibangun sebagai tugas kuliah Pengolahan Citra Digital.

## ✨ Fitur Utama

| Fitur | Deskripsi |
|-------|------------|
| 📂 **Load Gambar** | Upload gambar format JPG/PNG |
| 🔄 **Transformasi Geometri** | Rotasi, translasi, skala, flip |
| ✨ **Enhancement** | Brightness, contrast, sharpening |
| 📊 **Histogram** | Visualisasi histogram & equalization |
| 🎛️ **Noise & Filtering** | Gaussian/Salt & Pepper noise, Median/Gaussian filter |
| 🌊 **FFT** | Analisis frekuensi & inverse FFT |

## 🛠️ Teknologi yang Digunakan

- [Streamlit](https://streamlit.io/) - Framework web Python
- [OpenCV](https://opencv.org/) - Operasi citra digital
- [NumPy](https://numpy.org/) - Manipulasi array numerik
- [Pillow](https://python-pillow.org/) - Enhancement dasar
- [Matplotlib](https://matplotlib.org/) - Visualisasi histogram
- [SciPy](https://scipy.org/) - Fast Fourier Transform
